# Local-Store-Sales-Analysis
# 🛒 Local Retail Store Sales & Inventory Analysis

##  Project Overview
This project analyzes sales and inventory data of a local retail store using Python, SQL, Excel, and Power BI. The objective is to identify sales trends, monitor inventory levels, and create an interactive dashboard to support business decision-making.

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL
- Microsoft Excel
- Power BI

## 📂 Project Files
- `analysis.py – Python script for data cleaning and analysis
- `store_analysis_queries.sql – SQL queries for data analysis
- `local_store_sales_dataset.csv – Original dataset
- `local_store_sales_cleaned.csv – Cleaned dataset
- `Store_sales_analysis_dashboard.pbix – Power BI dashboard
- `Figure_1.png– Dashboard screenshot

## 📊 Dashboard Features
- Total Revenue
- Total Profit
- Total Orders
- Revenue by Category
- Revenue by Employee
- Revenue by Product
- Payment Method Analysis
- Low Stock Monitoring
  
 ##  Dashboard Preview
[Retail Sales & Inventory Dashboard](dashboard.png)

## Key Insights
- Grocery category generated the highest revenue.
- Rice 5kg is the highest revenue-generating product.
- Revenue was analyzed by employee performance.
- Payment methods (Cash, Card, and UPI) were compared.
- Products with stock remaining below 20 units were identified.

## Project Outcome
This project demonstrates end-to-end data analytics skills, including data cleaning, SQL querying, data visualization, and dashboard creation using Power BI.
